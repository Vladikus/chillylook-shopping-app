## Products Overview
* Products are displayed on the main screen as a greed with clickable icons to favorite the product or add to the cart.
* *__CircularIndicator__ is being displayed on the screen while products are loading or while any http request is being processed.*
* AppBar contains:
  * AppDrawer to display the navigation menu 
  * Filter to sort all products and your favorite products
  * Cart; There is a SnakcBar appearing at the bottom when adding a product to the cart with UNDO functionality

## Cart
